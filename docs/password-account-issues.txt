IT SUPPORT KNOWLEDGE BASE — PASSWORD & ACCOUNT ISSUES
======================================================

ISSUE: User cannot log in / forgot password
SOLUTION:
1. Go to https://password.company.com/reset
2. Enter your corporate email address
3. Check your email for the reset link (expires in 15 minutes)
4. Create a new password — minimum 12 characters, must include uppercase, lowercase, number, and special character
5. Log in with new password
6. If issue persists, call IT helpdesk: ext 4357

ISSUE: Account locked after too many failed attempts
SOLUTION:
1. Wait 30 minutes for automatic unlock
2. Or call IT helpdesk (ext 4357) for immediate unlock
3. Helpdesk will verify your identity using employee ID
4. Account unlocked within 5 minutes

ISSUE: Multi-factor authentication (MFA) not working
SOLUTION:
1. Ensure your phone time is synced correctly
2. Try the backup code from your MFA setup email
3. If phone is lost: call IT helpdesk with employee ID for emergency bypass
4. New MFA device setup takes 10 minutes with IT helpdesk

ISSUE: Okta SSO not working
SOLUTION:
1. Clear browser cache and cookies
2. Try incognito/private browser window
3. Check Okta status at status.okta.com
4. Reinstall Okta app on mobile if using mobile authenticator
5. Escalate to IT helpdesk if issue persists more than 30 minutes

ISSUE: VPN connection failing
SOLUTION:
1. Check internet connection first
2. Disconnect and reconnect VPN
3. Restart the VPN client (Cisco AnyConnect / GlobalProtect)
4. Try alternate VPN gateway: vpn2.company.com
5. Check if VPN client needs update — IT pushes updates every quarter
6. If still failing, run: ping vpn.company.com to test connectivity
